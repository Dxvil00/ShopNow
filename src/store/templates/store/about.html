Project Background

This web application is part of our midterm project for the subject Software Design.

It simulates a basic online store where users can browse products, manage a cart, and explore user-friendly UI/UX principles. The main goal is to demonstrate good software structure, team collaboration, and responsive design practices.

Technical Stack:
  • Backend: Django
  • Frontend: HTML, CSS, JavaScript, and Bootstrap
  • Database: Configured in Django (SQLite by default)
