About This Project

A student-led project created for our Software Design course, aimed at applying real-world development skills in an eCommerce context.

Project Background

This web application is part of our midterm project for the subject Software Design. It simulates a basic online store where users can browse products, manage a cart, and explore user-friendly UI/UX principles. The main goal is to demonstrate good software structure, team collaboration, and responsive design practices.

Educational Goals

Apply the principles of software design including modularity, usability, and maintainability within a full-stack web application.

Development Goals

Build a clean, responsive UI that reflects real-world applications using HTML, CSS (Bootstrap), Django, and modern UX principles.
